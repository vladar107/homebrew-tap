# Homebrew Tap — Claudescope

Homebrew formula for [Claudescope](https://github.com/vladar107/claudescope), a local, read-only viewer for your AI coding-agent transcripts (Claude Code, Codex, Junie).

## Install

```sh
  brew install vladar107/tap/claudescope
  claudescope                   # starts the app in the background and opens your browser
```

Or tap first, then install:

```sh
  brew tap vladar107/tap
  brew install claudescope
```

## Update

```sh
  brew upgrade vladar107/tap/claudescope
```

## Maintenance

Formula/claudescope.rb is generated and pushed automatically by the Claudescope release workflow on each version tag — don't edit it by hand.
